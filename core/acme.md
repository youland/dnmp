# 🔒 SSL 证书自动化 (ACME)

> **🎯 极客箴言**：在明文裸奔的时代，HTTPS 是 Web 引擎最基础的尊严。

本模块内置了 ACME 自动化脚本，智能双模签发 `Let's Encrypt` 绿锁证书，并与 Nginx 底层配置无缝对接。

### <span style="color: #1e90ff;">✒️ 一键签发与部署</span>

在 SSH 终端里，直接输入以下命令（将 `<你的域名>` 替换为实际域名）：
```bash
bash dnmp acme aa.bb.com
```

**⚠️ 防坑预警（执行前必读！）**
* **DNS 解析的时间差**  
在敲下回车之前，你**必须**确保该域名的IP已经成功解析并指向了本服务器。由于全球DNS节点生效存在延迟（通常需要 1~10 分钟），请不要在刚配置完解析的下一秒就立即执行脚本！

* **绝对禁止“暴力重试”**    
如果脚本提示签发失败（通常是因为 DNS 未生效），请务必等待 5 分钟后再试。`Let's Encrypt` 官方有极其严格的 API 请求频率限制（Rate Limit）。如果你短时间内疯狂重试失败，你的域名将被官方拉黑封禁整整 7 天！

**📂 签发后的资产交**  
当终端提示签发成功后，脚本已经为你全自动完成了以下操作：
*  **⚡ Nginx 配置自动装载**  
脚本已为你自动生成了该域名的 Nginx 独立配置文件：
配置文件路径：`/etc/nginx/sites-enabled/aa.bb.com.conf`  
（提示：脚本已默认帮你配置了 HTTP 强制跳转 HTTPS (301 Redirect)，后续网站的伪静态等高级配置，均在此文件内修改。）

*  **🔑 SSL证书私钥与公钥**  
你的 SSL 证书已安全入库，默认存放路径为：`/etc/nginx/ssl/aa.bb.com/`。
该目录下包含两把核心钥匙：  
> 私钥证书 `(Private Key)`：`aa.bb.com.key` （绝对机密，请勿泄露）  
> 公钥证书 ` (Fullchain)`：`aa.bb.com.cer`

*  **🔄 永久自动续签**  
Let's Encrypt 证书的物理有效期为 90 天。但你无须理会，也无须任何手动操作。ACME 自动脚本已向 Linux 系统的 cron 定时任务中注入了守护进程，它会在证书到期前自动与官方节点通讯完成静默续签，并平滑重载 Nginx，保障你的绿锁SSL证书永久生效。
